# 🧠 Cognitive Math Twin  
Digital Cognitive Twin for Math Students — Powered by AI

هذا المشروع يبني "توأم عقلي" للطالب في الرياضيات باستخدام:
- نماذج ذكاء اصطناعي (RNN, Transformers)
- تحليل أخطاء الطالب
- تتبع المعرفة Knowledge Tracing
- كشف الحالة الانفعالية (خوف، إحباط، توتر)
- توليد مسار تعلم شخصي Adaptive Learning Path

---

## 🚀 Features

### 1) Cognitive Modeling  
تحليل كامل لعقل الطالب:
- نقاط الضعف
- المفاهيم الخاطئة
- أنماط التفكير
- مستوى الإتقان

### 2) Error Prediction Engine  
يتوقع الخطأ قبل أن يحدث:
- نوع الخطأ
- احتمالية حدوثه
- سبب الخطأ

### 3) Adaptive Path Generator  
يبني مسار تعلم شخصي:
- تمارين مناسبة
- صعوبة تدريجية
- مواضيع مستهدفة

### 4) Cognitive Growth Tracker  
يتابع تطور الطالب:
- الدقة
- التحسن عبر الزمن
- مقارنة البداية بالنهاية

### 5) Math Fear Detector  
يكشف:
- الخوف
- الإحباط
- التوتر
- الثقة
